Dhaani: Human-AI Collaborative Software Development

Dhaani uses a 2-step approach: design first, then code. It combines human creativity with AI speed and knowledge to build modular software with human-in-the-loop control.

Created during research in interactive structured induction of programs for program synthesis, Dhaani emphasizes accountability, transparency, and maintainable code generation through structured interactions.

It is designed for scientific workflows, data science, AI-ML workflows, data analysis, and research challenges.
